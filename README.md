# Chi-Squared-Calculator
This calculator is avaliable in Excel or Google Sheets format and takes in a minimum of a 2 by 2 table and up to a 10 by 10 table! It can calculate the expected values, chi-squared value contributions, the chi-squared, and the degree of freedom!

# Steps for Use (applies for both Excel and Sheets formats)
Go to the **Step 1** and enter in your table. The calculator accepts a minimum of a $2 \times 2$ table up to a $10 \times 10$ table. This is the only editing you will need to do on this calculator. **Step 2** contains all the Expected Values for each value correlates to the position of each value in the table in **Step 1**. **Step 3** gives you the Chi-Squared contributions that correlates to the position of each value in the table in **Step 1**. **Step 4** contains the Chi-Squared, highlighted in black is your Chi-Squared value. **Step 5** contains the degrees of freedom, you degree of freedom is in the last cell in the bottom right corner.
